Get Started

Here is instructions on how to install:

GNU/Linux


Ubuntu
The package for Ubuntu is available via our Launchpad PPA.
Read more in the Ubuntu wiki page

Debian
Debian users can install Persepolis using standalone .deb packages.
See the Debian wiki guide for details

Arch Linux
Available in both official repositories and as the latest git version in AUR.
Follow the Arch guide on our wiki

Fedora
Persepolis is included in Fedora's official repositories.
Learn more in the Fedora section of our wiki

openSUSE
Installation instructions for openSUSE are available with step-by-step guidance.
Check the openSUSE wiki page

Bundle for all Linux distributions
A portable bundle for all Linux distributions is available on the releases page.
See more details on the bundle in the wiki

BSD

Visit wiki for learning how to install Persepolis on FreeBSD


MacOS

To install Persepolis on MacOS, you need to just simply download the latest package from the releases page then copy the .app file to your Applications folder!

more details on persepolis wiki


Microsoft Windows

Simply download and run Persepolis setup according to your system architecture
follow wiki for more details
